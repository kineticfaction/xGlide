xGlide a XML based MVC/MVP framework for PHP

DESCRIPTION:
xGlide is a PHP framework that makes rapidly developing MVC/MVP
applications a breeze. It was designed from the ground up to use
XML and REST to display data and works beautifully with RESTful
javascript framworks like ExtJS.
